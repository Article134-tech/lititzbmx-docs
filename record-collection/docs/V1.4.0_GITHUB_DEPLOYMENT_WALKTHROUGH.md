# v1.4.0 GitHub Deployment Walkthrough

## Recommended patch deployment

1. Extract `GH-Floored-v1.4.0-patch.zip` to a short local path.
2. Copy the contained `record-collection` folder into the repository root beside the existing `record-collection` folder.
3. Choose **Replace the files in the destination**.
4. Review the changed files in GitHub Desktop.
5. Commit with: `Add Greg Hill Floored 25-Short brochure archive`
6. Push origin.

## Full replacement alternative

The complete `Lititz-BMX-Record-Collection-v1.4.0.zip` may replace the existing `record-collection` folder in one controlled operation.
