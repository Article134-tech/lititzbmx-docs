# Visual Enhancement Change Report

**Current release:** v1.1.0  
**Updated:** 2026-07-21

The v1.1.0 presentation build replaces the seven-dossier access layer with a complete 21-dossier visual system.

## Changes

- rebuilt the main Record Collection banner;
- rebuilt all three category covers as complete visual indexes;
- created one consistent 16:9 repository navigation thumbnail for each dossier;
- updated collection and category README navigation;
- documented every later-created visual as a repository navigation derivative or composite.

## Preservation control

No original title card, supplied publication image, source transcript, or dossier evidence file was altered. Generated navigation assets are stored separately and are never described as original publication graphics.
