# File Naming Standard

## Record IDs

Use lower-case kebab-case and include the collection abbreviation when useful.

Examples:

- `fbc-001-damian-x-fulton`
- `fbc-greg-hill-floored`
- `fbc-004-greg-mathias-chasing-harry-hof`

Do not invent episode numbers when the source materials do not establish them.

## Role-based source filenames

Prefer:

- `podcast-proposal-public-redacted.pdf`
- `episode-outline.pdf`
- `press-release.pdf`
- `reels-and-short-descriptions.pdf`
- `original-transcript-timestamped.txt`

The exact original upload filename is preserved in `docs/source-inventory.md` and the private supplement.
