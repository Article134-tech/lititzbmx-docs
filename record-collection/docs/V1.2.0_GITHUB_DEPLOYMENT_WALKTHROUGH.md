# v1.2.0 GitHub Deployment Walkthrough

## Recommended package

Use `lititz-bmx-record-collection-v1.2.0-final-deployment.zip`.

## Before copying files

1. Open GitHub Desktop.
2. Confirm repository: `Article134-tech/lititzbmx-docs`.
3. Confirm branch: `main`.
4. Click **Fetch origin** and pull any available remote changes.
5. Make a temporary backup of the current local `record-collection` folder.

## Replace the controlled folder

1. Extract the deployment ZIP.
2. Confirm it contains one top-level folder named `record-collection`.
3. Copy that folder into the root of the local `lititzbmx-docs` repository.
4. Choose **Replace the files in the destination**.
5. Do not delete or replace any other top-level repository folder.

## Review in GitHub Desktop

Confirm:

- every changed path begins with `record-collection/`;
- one new dossier appears under `collections/pump-track-chat/records/`;
- the Pump Track Chat category count changes to 5;
- the complete collection count changes to 22;
- no unrelated campaign, learning-resource, brand-asset, standards, or root repository file changes.

Recommended commit summary:

`Add Pump Track Chat 2 dossier to Record Collection`

Commit to `main`, then click **Push origin**.
