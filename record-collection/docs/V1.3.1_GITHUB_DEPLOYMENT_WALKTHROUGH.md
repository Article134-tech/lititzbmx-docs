# v1.3.1 GitHub Visual-Fix Walkthrough

## Fast patch method

1. Download `GM4-visual-fix-v1.3.1.zip`.
2. Move it to `C:\` and rename it `GM4FIX.zip` if Windows reports a long-path problem.
3. Extract to `C:\GM4FIX`.
4. Open the local `lititzbmx-docs` repository folder.
5. Copy `C:\GM4FIX\record-collection` into the repository root beside the existing `record-collection` folder.
6. Choose **Replace the files in the destination**.
7. In GitHub Desktop, confirm all changed paths start with `record-collection/`.
8. Commit with: `Add visual access to Greg Mathias Episode 4 Shorts archive`.
9. Push origin.
