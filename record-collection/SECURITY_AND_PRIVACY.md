# Security and Privacy

This public repository was deliberately separated from a **private preservation supplement**.

## Public repository

The public package contains:

- descriptive records and metadata;
- public recording links and supplied public descriptions;
- title cards;
- public-video machine transcripts;
- public-access copies of proposals and press releases;
- production and derivative documents that passed review.

## Private supplement

The private package preserves:

- unredacted originals containing a phone number or direct email address;
- exact original uploaded filenames;
- duplicate source captures;
- the unaltered Jessica Reveal clip-notes PDF containing an unrelated personal review on its final page;
- the legacy Damian starter-package ZIP;
- source hashes and transformation mapping.

## Public transformations

Transformations are never hidden:

- Damian Fulton proposal: phone number and direct email redacted.
- Greg Mathias proposal: phone number and direct email redacted.
- Damian Fulton press release: direct email redacted.
- Jessica Reveal clip notes: unrelated final page removed from the public copy; original retained privately.

## Publication rule

Do not place the private supplement in a public GitHub repository. Store it offline, in encrypted storage, or in a separate private repository with access controls.
