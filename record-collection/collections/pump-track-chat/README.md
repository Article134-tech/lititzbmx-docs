# Pump Track Chat

**Category status:** Established  
**Default dossier type:** Interview Dossier

Pump Track Chat records conversations about pump tracks, BMX access, planning, community development, riding, and advocacy.

## Current repository status

No complete Pump Track Chat source submission is included in repository version 1.0.1. The category is preserved now so future records enter a stable home without changing the collection architecture.

## Future dossier intake

A complete source package should be reviewed for:

- canonical recording link and public title;
- current published description;
- transcript or recording-derived notes;
- title card or thumbnail;
- proposal, outline, release, press material, or clip documentation when surviving;
- privacy and rights concerns;
- clear distinction between interview testimony and independently verified information.

Future records should use the [Interview Dossier template](../../templates/interview-dossier/README.md) unless the recording is primarily a presentation, field capture, or other event type.
