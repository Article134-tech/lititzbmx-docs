# Pump Track Chat

**Category status:** Established.  
**Default dossier type:** Interview Dossier.

Pump Track Chat records conversations about pump tracks, BMX access, planning, community development, riding, and advocacy. No complete source submission for a Pump Track Chat dossier is included in repository version 1.0.0.

Future records should use the Interview Dossier template unless the recording is primarily a presentation, field capture, or other event type.
