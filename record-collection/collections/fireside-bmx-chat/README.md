# Fireside BMX Chat

Fireside BMX Chat preserves conversations that place BMX history inside a human setting: a bonfire, garage, museum, bridge, family memorial, archery lesson, or recovered phone conversation.

The compiled records are Interview Dossiers even when their formats vary substantially.

`Chasing Harry` is documented as an explicit subseries only where the supplied publication record uses that name. Other Harry-focused interviews are related records but are not automatically relabeled as part of the subseries.
