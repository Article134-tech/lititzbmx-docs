# Fireside BMX Chat — Episode 1

## Damian X. Fulton: BMX, Radical Rick, 1980s Toys, Advertising, and Harry Leary

This directory is the archival and editorial working package for **Fireside BMX Chat, Episode 1**, hosted by Kyle A. Huffman with guest Damian X. Fulton.

**Primary source:** https://www.youtube.com/watch?v=vtVr6GBJtlM  
**Duration:** 34:18  
**Publisher:** Lititz BMX

## Archival purpose

The package preserves the original public recording, its supplied description, a captured machine transcript, a working transcript, descriptive metadata, topic indexing, and a plan for a future visually enhanced edition.

The recording remains the authoritative oral-history source. The transcript is an access and research aid, not a replacement for the recording.

## Package structure

- `assets/` — supplied thumbnail and future approved visual assets
- `source/` — original captured transcript and source-description material
- `transcript/` — working and later publication-ready transcript versions
- `docs/` — editorial notes, topic index, provenance, and visual-enhancement plan
- `metadata.json` — machine-readable record metadata

## Transcript status

The current transcript is machine-generated and requires speaker identification, timestamp alignment, punctuation, proper-name verification, uncertain-word marking, and a final fidelity review against the full recording.

## Visual enhancement principle

The enhanced edition should add documented context without replacing or altering the original interview. Every image, document, artifact, advertisement, or clip should be traceable to a source and clearly distinguished from the interview itself.
