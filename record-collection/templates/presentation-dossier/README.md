# Presentation Dossier Template

1. Complete `presentation-record.md`.
2. Complete `metadata.json` using the repository schema.
3. Preserve the original source reference and public description.
4. Add transcript layers only when applicable.
5. Document missing material, privacy restrictions, and transformations.


## Public landing-page requirement

Before a completed Presentation Dossier is published, replace this template README with a visually accessible landing page that follows [the Lititz BMX GitHub Visual Presentation Standard](../../docs/VISUAL_PRESENTATION_STANDARD.md). Display the approved original title graphic when available, provide canonical-recording access, summarize the record, explain its significance, and clearly label any later-created repository cover.
