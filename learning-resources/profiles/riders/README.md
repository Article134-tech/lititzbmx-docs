# Rider Profiles

<img src="source/index.png" alt="Lititz BMX Rider Profiles visual index" width="900">

Ten independent rider-profile records preserve the supplied public-page image and text together. Each profile includes a source-bound overview, exact transcription, qualification notes, metadata and stable navigation.

> **Catalog boundary:** A rider profile is a learning-resource record. It does not become an artifact record, collection record or ownership record merely because the same rider appears elsewhere in Lititz BMX.
