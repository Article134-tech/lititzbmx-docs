# Brand Profiles

<img src="source/index.png" alt="Lititz BMX Brand Profiles visual index" width="900">

Four independent brand and publication profiles preserve the public source image and text as separate catalog records.

> **Catalog boundary:** Visual appearance is not shared catalog identity. The Bicycles & Dirt Magazine cover’s depiction of Debbi Kalsow is documented as a content appearance only; the magazine and rider profiles remain independent records.
