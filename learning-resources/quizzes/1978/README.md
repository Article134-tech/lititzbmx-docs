**Archive package version:** 1.1  
**Prepared:** July 19, 2026  
**Resource type:** Quiz and supporting historical article

This GitHub-ready package preserves the 1978 entry in the Lititz BMX history-quiz learning series under the same locked archival workflow used for the earlier Lititz BMX GitHub preservation projects:

- preserve the published interface and source sequence;
- preserve original wording separately from verified answers and normalized archival notes;
- transcribe the quiz and supporting article;
- verify every answer against the supplied evidence;
- document errors and discrepancies rather than silently correcting them;
- map every supplied image to a stable archive filename; and
- record SHA-256 fixity information.

## Critical verification findings

### Question 2 contains a published surname error

The quiz spells the rider’s name **Jeff Ruimner**. The supporting article spells it **Jeff Ruminer**.

The package preserves both source forms and prominently documents the discrepancy. It does not silently alter the published quiz.

### Extra-credit option D contains a published place-name spelling

The quiz publishes **Pittsburg, PA**. The Pennsylvania city is normally spelled **Pittsburgh**. Because it is a distractor, it is preserved exactly and documented.

### All answers are supported

All five main questions and the extra-credit question are answerable from the supplied two-page article spread. No answer was guessed and no external answer key was invented.
