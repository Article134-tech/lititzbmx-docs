## Critical verification findings

### The source crop is preserved, not hidden

The top of the supplied right-hand page is missing. The archive marks every affected passage and does not manufacture the absent wording.

### Cruiser-points wording receives a scope note

The quiz answer is **True** in the 1981 Pro Cruiser points context. Cruiser racing existed before 1981, so the answer is not presented as the first existence of any cruiser class.

### JMC is retained with contextual evidence

The surviving source fragment connects JMC with the stylish-rider discussion associated with the pants question, but the sentence opening is cropped.

### Bob Morgan evidence is separated by strength

“Trash Can” Bob Morgan is supported by a period BMX Action article index. The no-footer/one-hander answer is retained as Bob Morgan with an explicit cropped-source warning.
