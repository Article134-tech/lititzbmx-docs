# BMX History Quiz: 1980

## Archived learning resource

**Live learning page:** https://sites.google.com/view/lititzbmxinventorylist/learning-resources/quizzes/1980-bmx-history-quiz  
**Archive package version:** 1.0  
**Prepared:** July 19, 2026  
**Resource type:** Quiz and supporting historical article

This GitHub-ready package preserves the 1980 entry in the Lititz BMX history-quiz learning series under the same locked archival workflow used for the preceding Lititz BMX GitHub preservation projects:

- preserve the published interface and source sequence;
- preserve original wording separately from verified answers and archival findings;
- transcribe the complete quiz and supporting article;
- verify every answer against identified evidence;
- document misspellings, wordplay, source gaps, and supplemental verification rather than disguising them;
- map every supplied image to a stable archive filename; and
- record SHA-256 fixity information.

## Critical verification findings

### “PRO-BABLY” is preserved

The quiz prints **PRO-BABLY**; the supporting article prints **probably**.

### “Avalance” is a published quiz error

The quiz prints **The Avalance**; the article caption prints **The Avalanche**.

### The Bobby Encinas statement is false

The supplied retrospective identifies Stu Thomsen as winning his first NORA Cup during 1980. A published BMX Action historical listing also records Stu Thomsen for 1980.

### Mongoose required supplemental verification

The supplied retrospective does not identify the NORA Cup No. 1 Team. A published BMX Action historical listing records **Mongoose** as the 1980 team. The source URL is preserved in the answer key, ledger, and source index.

### All answers are documented

Seven answers are supported directly or contextually by the supplied retrospective. One answer is supported by a separately identified published BMX Action source. No answer was invented.
