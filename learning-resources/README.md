# Lititz BMX Learning Resources

This directory contains interactive educational materials produced by Lititz BMX to connect puzzles, profiles, images, artifacts, people, places, and historical subjects with the broader archive.

---

## Current Learning Resources

### [Lititz BMX Profiles](profiles/)

**People, Makers and Places — A Visual Learning Atlas**

A source-preservation and visual-learning archive documenting **10 rider profiles**, **four independent brand and publication profiles**, and **150 track entries** across all 10 published Track Profiles pages.

The archive preserves the supplied source images, exact published text, complete page captures, structured registers, metadata, documentation, source qualifications, relationship boundaries, and SHA-256 fixity records.

Each rider, brand, publication, and track remains an independent catalog record. Visual or contextual connections between records support discovery and interpretation without merging their identities, provenance, ownership, or historical meaning.

**Status:** Complete source-preservation archive v1.0.0 / 37 supplied source images preserved / 0 pending-source records

[Open the Lititz BMX Profiles visual learning atlas](profiles/)
