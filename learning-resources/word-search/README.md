# Interactive BMX Word Search

## Click a term. Learn the history behind it.

**Live interactive resource:** https://sites.google.com/view/lititzbmxinventorylist/learning-resources/word-search  
**Archive package version:** 1.0  
**Prepared:** July 19, 2026  

This GitHub-ready package preserves the Lititz BMX interactive word-search learning resource using the governing model established for **#RebuildRadicalRick**:

- preserve the published sequence and interface relationships;
- preserve original wording separately from normalized archival summaries;
- document discrepancies instead of correcting them silently;
- map every supplied page, image, and destination URL to a stable record; and
- maintain fixity information for the archived files.

## Critical verification findings

- `CUP` is published in the answer list but is not present in the supplied grid.
- `NORA` appears twice; `GT` appears five times.
- No intended occurrence is selected for NORA or GT without an official answer key.
- The Miranda entry is intentionally fictional alternate history and is labeled accordingly.
- The live Diamondback page uses the working slug `diamonback-word-search`; that exact URL is preserved.
