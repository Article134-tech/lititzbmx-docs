# Lititz BMX Standards

This directory contains public documentation describing the standards, conventions, and workflows used to support the Lititz BMX Archive.

The goal of these documents is consistency across archive pages, podcasts, educational resources, rider profiles, artifact records, and media productions.

These standards continue to evolve as the archive grows.
