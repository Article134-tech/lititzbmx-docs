## The story

This 1st Place plaque from a Cactus Park BMX State Qualifier features artwork by Radical Rick creator Damian Fulton. It was the first item acquired from the “Leary Locker” collection, which ultimately led to the discovery of additional artifacts connected to BMX legend Harry Leary. The plaque also inspired a podcast conversation with Fulton and a charity effort benefiting Samaritan’s Purse, where all proceeds were donated.
