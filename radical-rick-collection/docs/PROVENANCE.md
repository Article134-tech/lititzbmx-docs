# Provenance and Source Handling

Volume One combines four evidence layers:

1. **Supplied artifact photographs** — preserved byte-for-byte in `source/`.
2. **LititzBMX.com collection wording** — preserved as `source_title` and through two source-page captures.
3. **Lititz BMX inventory records** — used for normalized titles, provenance, significance, edition and source links where available.
4. **Existing repository dossiers and campaign archives** — linked as canonical media and process records rather than copied into the collection.

No social-media caption, product description or inventory statement is silently converted into independent verification. Attributed claims remain attributed. Observable object details are separated from narrative significance and unresolved questions.
