# Known Discrepancies and Normalized Wording

This document preserves differences between the original Google Sites captions, visible object wording and the normalized Harry’s Room record layer.

## 26.0067 — legacy identifier 67

The original source displayed **67**. Harry’s Room uses the canonical form **26.0067** and preserves **67** as a legacy identifier.

## 26.0053 — Race / Racers Against Drugs

The physical plaque reads **RACE AGAINST DRUGS**. The original collection caption used **Racers Against Drugs**. The record title follows the object wording and preserves the caption wording as an alternate title.

The original caption also used **Article ID** instead of **Artifact ID**. Harry’s Room normalizes the field name while documenting the typo.

## 26.0023 — Peak Performance

The original collection caption contained **PERFORMNCE**. The normalized public title uses **Peak Performance**.

## 26.0048 — repeated across two source pages

The F-1 Challenge Trophy appeared on both original source pages. It is one accession and therefore has one integrated record with both source-page references.

## 26.0048 — F1 / F-1

Both forms appear in the supplied source wording. The record preserves both rather than forcing an unsupported single historical spelling.

## 26.0045 — research post versus verified finding

The supplied GHP-bars post contains approximate dates, team-history questions and a statement connecting Greg Hill to the origin of “The Leary” jump. Harry’s Room preserves that post as research evidence but does not convert those statements into verified artifact findings.

## 26.0030 — helmet identification

The helmet identification is based on the collection’s visual comparison with footage or photographs. It remains a collection attribution rather than an independent authentication.

## 26.0076 — direct image not supplied
