# Original Materials Policy

The graphic-novel presentation is created through sequencing, headings, tables, captions and links. It does **not** create replacement art.

- No generated artwork is included.
- No source artifact image is retouched, stylized or composited.
- All 18 visual files are preserved from the user-supplied source set without pixel alteration.
- Page-capture screenshots remain evidence files and are not treated as separate artifacts.
- Descriptive language distinguishes the physical object, the published source wording and the curatorial interpretation.
