# Archival Notes

## Record boundary

The campaign root and each child page are registered independently. A shared theme, repeated image, artist, song or rider does not merge records.

## Visual preservation

All supplied image files were copied byte-for-byte into the archive. No color correction, cropping, sharpening, retouching or recomposition was applied to the preserved files.

The root display uses the supplied Google Sites campaign index as its hero rather than generating replacement artwork.

## Text treatment

Campaign wording is preserved from the text supplied by the archive owner and from visible source evidence. Obvious grammar, punctuation and capitalization differences remain intact. Archival explanation is separated from the published wording.

## Ordering

The track order uses the reconstructed campaign page ledger:

- campaign root as Track 01 / Intro
- 22 child pages in sequence
- Curtain Call as Track 23 / Outro

The mixed-tape framing is a GitHub presentation layer. It does not claim that the original campaign distributed an audio compilation.

## External media

The archive stores the published YouTube URLs and visible artist/song labels. It does not download, mirror or redistribute the linked audio or video.
