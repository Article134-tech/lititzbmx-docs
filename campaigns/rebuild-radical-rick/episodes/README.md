# Rebuild Radical Rick Episode Index

This directory contains the individual archival records for the Lititz BMX **#RebuildRadicalRick** campaign.

The campaign contains **19 officially numbered episode positions**:

- **18 distinct episodes have been verified**
- **Episode 15 is preserved as an unverified lost episode**
- The official numbering and verified publication chronology do not always match
- Missing material has not been reconstructed through assumption

## Archival Method

Each episode record separates:

1. original campaign language;
2. later archival summaries and context;
3. source and verification notes;
4. unresolved questions or missing material.

Unverified information is identified openly rather than completed through assumption.
