# Lititz BMX Public Inventory

This directory contains a public snapshot of the Lititz BMX artifact inventory.

## View the Inventory

- [Browse the searchable CSV inventory](Lititz-BMX-Inventory.csv)
- [View the illustrated PDF inventory](Lititz-BMX-Inventory.pdf)
- [Visit the live Lititz BMX Archive](https://sites.google.com/view/lititzbmxinventorylist)

## About the Inventory

The inventory documents bicycles, frames, jerseys, helmets, awards, artwork, publications, photographs, equipment, and other materials connected to BMX history.

The CSV edition provides searchable, downloadable, machine-readable inventory data. Embedded Google Sheets images are not included in the CSV export. The PDF edition preserves the illustrated layout and artifact photographs.

Artifact identifiers follow [LBS-002 — Artifact Numbering Standards](../standards/LBS-002-Artifact-Numbering-Standards.md).

The live Lititz BMX Archive remains the primary source for current artifact pages, related media, and additional historical context. The files in this directory represent a dated public snapshot and may be updated as cataloging continues.

## Public Use

These files are published for research, education, discovery, and BMX preservation. Lititz BMX names, branding, archive content, artifact records, and archive links remain the property of Lititz BMX.
